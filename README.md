# Dynamic Employee Filter System

A reusable and configuration-driven employee filtering application built with React, TypeScript, Vite, and Material UI.

## Features

* Dynamic filter builder
* AND / OR filter logic
* Configuration-driven architecture
* Nested object filtering
* Material UI DataGrid
* Pagination and sorting
* Mock employee dataset
* TypeScript support

## Tech Stack

* React
* TypeScript
* Vite
* Material UI
* MUI DataGrid

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
